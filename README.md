# Issak Gezehei - Portfolio Website

**Live Site:** [https://igezehei.github.io](https://igezehei.github.io)

This is my personal portfolio showcasing my work as a Product Leader, SaaS Innovator, and AI Strategist. The site includes insights into AI trends, emerging technologies, and my professional experience.

## 🚀 Features

- Responsive design with Tailwind CSS
- Auth0 authentication integration (optional)
- Interactive AI chatbot integration (configurable)
- Mobile-friendly navigation
- Modern UI with smooth animations

## 🔧 Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/igezehei/igezehei.github.io.git
   cd igezehei.github.io
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   python3 -m http.server 8000
   # or
   npx serve
   ```

3. (Optional) To enable Auth0 authentication, create `auth-config.json`:
   ```bash
   cp auth-config.example.json auth-config.json
   # Edit auth-config.json with your Auth0 credentials
   ```

## 🔒 Security

This is a **public repository**. No secrets or credentials are committed to version control.
